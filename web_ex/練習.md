# 網頁設計
## div練習
```
<!doctype html>
<html>
  <head>
    <meta charset="utf-8">
    <title>示範div區塊</title>
  </head>
  <body>
    <div>
      <h1>我的程式集</h1>
      <ul>
        <li><a href="br.html">換行練習</a></li>
        <li><a href="超連結.html">超連結練習</a></li>
        <li><a href="article.html">文檔獨立練習</a></li>		
      </ul>
    </div>
  </body>
</html>
```
